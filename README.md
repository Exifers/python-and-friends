- [marshmallow](https://marshmallow.readthedocs.io/en/latest/) : ORM/ODM/framework-agnostic library for converting complex datatypes, such as objects, to and from native Python datatypes

## Python tricks
- strip and rstrip
```python
data = 'rgba(255,0,0)'
data.strip('rgb(').rstrip(')')
print(data)
# 255,0,0
```
